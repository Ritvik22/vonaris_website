# Web3Forms Setup Instructions

## Get Your Access Key

1. Go to https://web3forms.com
2. Enter ritvikshah2009@gmail.com
3. Click "Create Access Key"
4. Copy the access key you receive

## Update the Code

Replace `YOUR_WEB3FORMS_ACCESS_KEY` in both files:
- `/src/pages/Contact.jsx` (line 15)
- `/src/pages/Careers.jsx` (line 15)

## Benefits of Web3Forms

- ✅ Free forever
- ✅ No branding
- ✅ Custom redirects work perfectly
- ✅ Unlimited submissions
- ✅ Spam protection built-in
- ✅ Email notifications to ritvikshah2009@gmail.com
- ✅ No thank you page from provider

## How It Works

1. User fills out form
2. Form is submitted via JavaScript fetch to Web3Forms API
3. Email is sent to ritvikshah2009@gmail.com
4. User is redirected to your custom `/thanks` page
5. No provider branding or default pages shown
